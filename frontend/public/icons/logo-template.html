📸 Cara Screenshot Logo AE

Screenshot logo di bawah dengan ukuran yang tertera, lalu simpan sebagai PNG.

  1. Klik kanan pada logo → Save Image As... atau Copy Image
  2. Untuk 192x192: Klik logo pertama, simpan sebagai icon-192.png
  3. Untuk 512x512: Klik logo kedua, simpan sebagai icon-512.png
  4. Letakkan di folder frontend/public/icons/
